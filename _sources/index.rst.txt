Usage
-----

To run the program simply run the executable file gdsimsapp from the command line. This will display a command-line interface which will provide instructions to follow. For Windows:

.. code-block:: C++
	gdsimsapp.exe

For those comfortable with C++ programming, a code-based option also exists.
Output files will be generated in an ``output_files`` subdirectory in the chosen running directory. 
More details can be found in the Vignettes and User Guide.
